# Changelog

## [5.0.1] - 2026-04-17
### Fixed
- Cursor flickering in fullscreen mode.
- Terminal "tearing" by implementing string buffering.

## [5.0.0] - 2026-04-16
### Added
- **ASCII Art Mode**: Added `-f` flag for big centred numbers using `figlet`.
- **Target Time**: Support for HH:MM format (e.g., `countdown 04:20`).
- **Auto-Hide Cursor**: Cursor now hides during fullscreen to improve UI.
